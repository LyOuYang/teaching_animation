# AP-CSP: Algorithms & Programming - 核心概念全景

> **课程愿景**：通过 8 个交互式微场景，让抽象的算法概念在物理世界中显形。
> **适用人群**：AP Computer Science Principles 学生 / 高中生
> **设计风格**：Cyberpunk / Sci-Fi / gamified

---

## 1. Introduction to Algorithms
**演示：The Robot Chef (机器人大厨)**
*   **概念**：算法是精确的、有序的指令序列。
*   **核心隐喻**：给完全不懂常识的机械臂下达做三明治的指令。
*   **交互**：
    *   拖拽指令卡片：[放面包]、[放火腿]、[盖面包]。
    *   **错误演示**：如果顺序是 [放火腿] -> [放面包]，火腿会掉在桌子上。
*   **Aha!时刻**：顺序颠倒会导致完全不同的非法结果。

## 2. Boolean Expressions
**演示：The Laser Heist (激光大盗)**
*   **概念**：`True`/`False`，逻辑运算符 (`AND`, `OR`, `NOT`)。
*   **核心隐喻**：博物馆的安保激光网。
*   **交互**：
    *   **AND Gate**：必须同时按下两个开关（双人协作）才能关闭激光。
    *   **OR Gate**：按下任意一个红色紧急按钮都能触发警报。
    *   **NOT Gate**：一个反相器，开关向上激光关，开关向下激光开。
*   **Aha!时刻**：复杂的安保系统是由简单的“是/否”开关逻辑堆叠而成的。

## 3. Variables and Assignments
**演示：Infinite Lockers (无限储物柜)**
*   **概念**：变量不仅仅是值，是有名字的容器。`Assign` 是覆盖操作。
*   **核心隐喻**：学校的储物柜。
*   **交互**：
    *   **声明**：给 101 号柜子贴上 `Score` 的标签。
    *   **赋值**：打开柜子，扔掉旧的作业本，放入新的 `100` 分试卷。
*   **Aha!时刻**：一个柜子同一时间只能装一样东西。新值进入，旧值立刻消失（被 GC 回收/垃圾桶）。

## 4. Strings
**演示：Neon Sign Maker (霓虹灯工坊)**
*   **概念**：字符串是字符的有序序列。拼接 (`Concatenation`) 与 子串 (`Substring`)。
*   **核心隐喻**：焊接霓虹灯管。
*   **交互**：
    *   **拼接**：将 "HELLO" 和 "WORLD" 两根灯管焊接在一起，变成更长的 "HELLOWORLD"。
    *   **索引**：每一个字母下面都有一个小尺子刻度（0, 1, 2...）。
    *   **截取**：用激光切割机切下索引 1 到 4 的部分。
*   **Aha!时刻**：空格（Space）也是一种看不见的玻璃灯管，占位置！

## 5. Data Abstraction
**演示：The Starship Cockpit (星际飞船驾驶舱)**
*   **概念**：隐藏复杂细节，只暴露简单接口。
*   **核心隐喻**：引擎室 vs 驾驶舱。
*   **交互**：
    *   **View A (引擎视角)**：满屏的电压表、油压管、报错代码（复杂 Implementation）。
    *   **View B (飞行员视角)**：只有一个巨大的红色按钮 "HYPERDRIVE"（简单 Interface）。
*   **Aha!时刻**：我不需要知道引擎怎么喷射燃料，我只需要按按钮。这就是抽象。

## 6. Lists
**演示：The DJ Playlist (DJ 播放列表)**
*   **概念**：有序集合、索引访问、动态增删。
*   **核心隐喻**：音乐播放列表。
*   **交互**：
    *   **Index**：点击 #3 直接切歌。
    *   **Insert**：在 #2 插队一首歌，观察 #3 及后面的歌全部自动“下移”一位。
    *   **Length**：歌单长度随操作实时变化。
*   **Aha!时刻**：插队（Insert）是很累的，后面所有人都得动！

## 7. Mathematical Expressions
**演示：The Logic Tree (算术树)**
*   **概念**：运算优先级、嵌套表达式。
*   **核心隐喻**：生长的果树。
*   **交互**：
    *   可视化表达式 `(3 + 5) * 2`。
    *   `3` 和 `5` 是果实，挂在 `+` 的树枝上。
    *   并不是从左读到右，而是先算树枝末端。
    *   `+` 结出一个 `8` 的大果实，掉落到 `*` 的树枝上与 `2` 结合。
*   **Aha!时刻**：数学不是一行扁平的字，它有立体的结构（树）。

## 8. Conditionals
**演示：Railway Switch (铁路道岔)**
*   **概念**：`If-Else` 分支逻辑。
*   **核心隐喻**：岔路口。
*   **交互**：
    *   火车轰隆隆开来。
    *   信号灯（Condition）：`Speed > 100`?
    *   **True**：道岔自动扳向右边（高速轨道）。
    *   **False**：道岔保持直行（普通轨道）。
*   **Aha!时刻**：代码的执行流在物理上被“掰弯”了。
